Описание
Cameras - это мобильное приложение для управления системой видеонаблюдения и доступом к дверям. 
Это приложение позволяет пользователям просматривать камеры в реальном времени, а также управлять доступом к дверям их дома или офиса.

Основные функции:
-Просмотр Камер: Возможность просматривать потоки с камер в реальном времени.
-Управление Дверями: Открывать и закрывать двери на расстоянии.
-Переименование Устройств: Изменение названий камер и дверей для удобства использования.
-Свайп-функционал: Удобное управление элементами интерфейса благодаря свайпам.
-Интеграция с Базой Данных: Локальное хранение информации о камерах и дверях.

Технологии
Проект использует следующие технологии и фреймворки:
Kotlin: Язык программирования для разработки Android приложений.
Jetpack Compose: Современный инструментарий для построения интерфейса в Android.
Realm Database: Локальная база данных для Android.
Coil: Библиотека для загрузки и кэширования изображений.
Network: KTOR
Kotlin Coroutines.
Паттерн проектирования: MVVM
